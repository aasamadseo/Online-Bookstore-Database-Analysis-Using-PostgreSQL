# Online-Bookstore-Database-Analysis-Using-PostgreSQL
Developed a relational database project for an Online Bookstore using PostgreSQL. The database consisted of three interconnected tables: Books, Customers, and Orders. Imported and managed data from CSV files, established primary and foreign key relationships, and performed SQL-based analysis to generate business insights.
Key activities included:

Database design and table creation
Data import and validation
Primary and foreign key implementation
SQL Joins (INNER, LEFT, RIGHT)
Aggregate functions (SUM, COUNT, AVG)
Filtering and sorting data
Customer purchase analysis
Book sales analysis
Order trend reporting

Skills: PostgreSQL, SQL, Database Design, Data Analysis, Data Cleaning, Joins, Aggregate Functions, Query Optimization
